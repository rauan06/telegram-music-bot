# Telegram music bot
# Usage:
Create your bot token API from https://t.me/BotFather \
\
Clone this repo: 
```bash
git clone https://github.com/rauan06/telegram-music-bot.git
```
\
Create new environmnet: 
```bash
python -m venv env
```
\
Export your API token: 
```bash
export BOT_TOKEN=YOUR_BOT_TOKEN
```
